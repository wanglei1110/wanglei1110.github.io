发表于2023-04-21|更新于2023-09-06
|阅读量:

使用存疑

  • 联合使用gpt-3.5-turbo&newbing报错：
    NEWBING_COOKIES未填写或有格式错误。

概述

一句话介绍该项目

由中科院研发并开源的github项目，基于chatgpt扩展的代码，当前功能主要用于学术研究和代码学习，通过前端展示及使用。简单修改代码可实现功能自定义。

镜像部署 （使用docker）

下载项目

git clone https://github.com/binary-husky/chatgpt_academic.git

cd chatgpt_academic

先注释掉Dockerfile的最后一行: 运行 main.py 的

修改config.py文件

使用

镜像及容器

  • 打镜像

docker build -t gpt-academic .

  • 启容器

把本地文件映射到/gpt/wl 运行且不启jupyter，这样便于修改文件内容；8812为内置的端口，对外映射1008

本地使用：
docker run -dit -v /NAS1/users/wl/wl:/opt/wl -p 1008:8812 gpt-academic:latest bash

云端使用：
腾讯云
docker run -dit -v /root/home/chatgpt_academic:/gpt/chatgpt_academic -p 80:80 gpt-academic:latest bash
AWS
docker run -dit -v /home/ec2-user/chatgpt_academic/:/gpt/ -p 80:80 -p 1000:22 gpt-academic:latest bash

  • DELL:
    docker run -dit -v /home/dw/snap/chatgpt_academic0727:/gpt/chatgpt_academic -p 8081:80 gpt-academic0629:latest bash

/home/chatgpt_academic

  • 运行

安装vim以防用到修改文件功能
apt-get update
apt-get install vim
cd chatgpt_academic

「说明」：运行 mian.py 前需要生成并修改根路径下的config_private.py文件，详细修改见下方[[04-Seminar/23-03/项目学习-中科院学术chatgpt_academic#修改配置(运行 main.py 前修改)]]

python main.py

在浏览器打开网址： 192.168.210:1008

使用以下操作在容器中执行ssh服务

apt-get update
apt-get install vim -y
apt-get install openssh-server
apt-get remove openssh-server # 移除ssh服务
sed -ri ‘s/#PermitRootLogin prohibit-password/PermitRootLogin yes/g’ /etc/ssh/sshd_config && echo ‘root:d’|chpasswd
EXPOSE 22 在docker run 的时候映射

修改配置(运行 main.py 前修改)

使用一个新的 config_private.py;将config.py内容复制再修改

程序概述:
配置API_KEY的设置和代理。API_KEY的常量来存储私人的API密钥。USE_PROXY的常量来标记是否需要使用代理。
如需代理，USE_PROXY设为True(默认为False) 则使用了一个名为proxies的字典来存储代理网络的地址，其中包括协议类型、地址和端口。

    # 修改内容：
    USE_PROXY = True
    API_KEY = "sk-xxx" （此key无效）修改成自己的key
    proxies = {
        #          [协议]://  [地址]  :[端口]
        "http":  [协议]://  [地址]  :[端口],
        "https": [协议]://  [地址]  :[端口],
    } # 修改代理网络
    WEB_PORT = 8812 # 修改端口

核心功能函数

core_functional.py 定义了一些核心功能函数，可以用户自定义结果。
每个功能都有一个Prefix属性和Suffix属性，Prefix是指在用户输入的任务前面要显示的文本，Suffix是指在任务后面要显示的文本。Color属性指示按钮的颜色，PreProcess函数表示对输入进行预处理的函数。
